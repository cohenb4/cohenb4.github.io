Steps of completing a Neural Network Project Here're the steps to complete a project, which we'll cover in the labs:
  1. Load data & Preprocess
  2. Build the model: Define and complie the model
    1. Define the model: Firstly we need to specify layer type, number of neurons in each layer, activation function, input shape.
    2. Compile the model: Next we need to specify optimizer, loss function and metrics:
      1. Loss function - measures how accurate the model is during training, we want to minimize this with the optimizer.
      2. Optimizer - how the model is updated based on the data it sees and its loss function.
      3. Metrics - used to monitor the training and testing steps.
      4. Train the model: Fit the model with training data
  3. Make predictions and Evaluate
